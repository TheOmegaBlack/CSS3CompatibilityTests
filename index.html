Properties which don't exist in IE Grid:

  • grid-area
    Used to give a name to an area
  • grid-template-areas
    Used to define the area on the grid
  • grid-auto-columns
    Used to tell the grid how to behave in case of elements positioned in columns out of the specified ones
  • grid-auto-rows
    Used to tell the grid how to behave in case elements positioned in rows out of the specified ones
  • grid-auto-flow
    Used to tell the grid how to autofill the empty cells
